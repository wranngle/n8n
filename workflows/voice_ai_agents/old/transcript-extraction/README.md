# Transcript Field Extractor - AI Agent Workflow

## Overview
Extracts structured fields from ElevenLabs voice conversation transcripts using a single bulk prompt to Gemini LLM.

## Business Process
- **Process**: voice_ai_agents
- **Type**: ai_agent
- **Phase**: DEV

## Architecture
```
┌─────────────┐     ┌───────────────────┐     ┌──────────────────┐
│   Webhook   │────▶│ Fetch Conversation│────▶│ Fetch Agent      │
│  (trigger)  │     │   (ElevenLabs)    │     │   Config         │
└─────────────┘     └───────────────────┘     └──────────────────┘
                                                      │
                                                      ▼
┌─────────────┐     ┌───────────────────┐     ┌──────────────────┐
│  Respond    │◀────│  Parse Response   │◀────│ Call Gemini      │
│ to Webhook  │     │    (extract)      │     │  (bulk prompt)   │
└─────────────┘     └───────────────────┘     └──────────────────┘
```

## API Endpoints Used

### ElevenLabs
- `GET /v1/convai/conversations/{conversation_id}` - Fetch transcript
- `GET /v1/convai/agents/{agent_id}` - Fetch agent config

### Gemini
- `POST /v1beta/models/gemini-3-pro:generateContent` - LLM extraction
